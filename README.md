# Cute Python Photobooth 🩷 

A simple photobooth application built with Python.

Features
- Live camera preview
- Photo capture
- Filters (Normal, Gray, Blur, Cartoon)
- Add stickers to photos
- Auto-cropped transparent stickers
- Download final image

Tech Used
- Python
- OpenCV
- Tkinter
- Pillow

A fun side project I built while experimenting with computer vision and GUI development.
